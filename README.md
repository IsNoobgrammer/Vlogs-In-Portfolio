# Vlogs-In-Portfolio
Vlogs written here will be reflected on my portfolio. Less hassle;

## Blog Format
To ensure your blogs are correctly fetched and displayed on the portfolio website (`shaurya.sharthak`), strictly follow these formatting rules:

- **Location**: Each blog must be its own folder located in the **root** of this repository. The portfolio fetches all root directories as blogs.
- **Title**: The folder name is automatically converted into the blog title (e.g., `my-first-post` becomes `My First Post`).
- **Content File**: The main content must be in a file named exactly `README.md` inside the blog's folder.
- **Date**: The `README.md` content must contain a date in the format `YYYY-MM-DD` anywhere in the text. This is used to sort the posts on the portfolio.
- **Excerpt**: The first paragraph of the `README.md` (excluding headings and frontmatter) is automatically used as the preview excerpt on the portfolio card.
- **Assets**: All images, videos, and other assets should be stored within the same folder and referenced relatively.

This structure ensures seamless integration with the portfolio's inline Markdown reader.
